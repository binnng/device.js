# device.js

## 特性

* IsTouch
* IsAndroid
* IsIPad
* IsIPhone
* IsIOS
* IsWinPhone
* IsWebapp
* IsXiaoMi
* IsUC
* IsWeixin
* IsBaiduBox
* IsBaiduBrowser
* IsChrome
* IsPC
* IsHTC
* IsBaiduWallet
* IsDebug
